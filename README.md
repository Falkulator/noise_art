# noise_art

Fun randomness. 


### Version
0.0.1

### Tech

* [PaperJS] - Awesomness
* [tinycolor] - Color conversions
* [Font Awesome] - Icons


### Installation

Install dependencies
```sh
$ npm install 
$ bower install
```
Run Gulp Server
```sh
$ gulp serve
```
